## What I actually build

**LeadForge** — AI-powered lead scoring and outreach engine
Pulls company and contact data, scores leads with ML, and runs automated
personalised outreach sequences. Built a full pipeline: enrichment →
scoring → email generation → tracking. Tech: Node.js, Python, Apollo API,
Claude API, PostgreSQL.
[github.com/aayush2724/LeadForge](https://github.com/aayush2724/LeadForge)

**Music Intelligence Engine** — Shazam-inspired audio analysis app
Identifies songs from audio input using ACRCloud fingerprinting, then runs
deep audio analysis (tempo, mood, spectral features) via librosa and YAMNet.
Full-stack: React/Vite frontend, Node.js gateway, Python/FastAPI ML service.
_In progress_

**Job Portal** — role-based hiring platform
Next.js app with NextAuth.js auth, MongoDB, and separate recruiter/candidate
flows. Built for production deployment on Vercel.
